Extractive_QA_Pipeline.ipynb содержит базовый алгоритм извлечения ответов из корпуса документов с иипользованием моделей, рекомендованных библиотекой haystack.

Haystack_hybryd_qa_test.ipynb содержит гибридную систему, в качестве моделей hf_transformers. Данный подход можно использовать для построения очень простых вопросно-ответных систем.

Llama_cpp_test_haystack.ipynb содержит систему построенную на Llama_cpp (сжатые GGUF модели). Работает локально, но довольно медленно и часто ошибается.

langchain_deep_seek_and_tools.ipynb содержит оптимальное решение. Работа мощной рассуждающей модели, навигация по датасету с помощью агента create_pandas_dataframe.
